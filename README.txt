Mobiquity One Demo Project

This GitHub-ready project contains:
- Source code for the Mobiquity One demo Android app
- Placeholder product images and description documents
- Codemagic CI/CD configuration file (.codemagic.yaml)

Instructions:
1. Push this folder to a GitHub repository.
2. Codemagic will detect the .codemagic.yaml file and build the APK.
3. Replace placeholder images and descriptions in assets/placeholders/ as needed.
